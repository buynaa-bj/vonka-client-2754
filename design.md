# Vonka Studio — Design System

## Brand
- **Name**: Vonka Studio
- **Tagline**: "Print the Future"
- **Industry**: 3D Printing Studio (Mongolia)

## Color Palette
- **Background**: `#080810` — near-black with blue-black tint
- **Surface**: `#10101a` — elevated panels
- **Surface 2**: `#16162a` — cards
- **Border**: `rgba(255,255,255,0.07)` — subtle glass borders
- **Primary**: `#c0c8e0` — metallic silver-blue
- **Accent**: `#7b8cff` — electric indigo/blue
- **Accent 2**: `#e8e0d0` — warm silver/titanium
- **Text**: `#f0f0f5` — near-white
- **Text muted**: `#8888a0` — gray-blue
- **Gradient**: from `#7b8cff` via `#a0b0ff` to `#c0c8e0` — metallic gradient

## Typography
- **Display**: `Orbitron` (Google Fonts) — futuristic, geometric — for headings, logo
- **Body**: `Syne` (Google Fonts) — clean, modern — for body text
- **Mono**: `JetBrains Mono` — for labels, specs, technical info

## Motion
- Page load: staggered fade+slide reveals
- Scroll: intersection observer animations
- 3D: cursor-reactive rotation, scroll-driven transforms
- Hover: metallic sheen transitions, scale micro-interactions
- Duration: 0.4s easing `cubic-bezier(0.16, 1, 0.3, 1)`

## Layout
- Max width: 1400px
- Sections: full-bleed backgrounds, asymmetric layouts
- Grid: CSS Grid with intentional asymmetry
- Spacing: generous negative space in hero, dense in product grid

## Visual Style
- Dark industrial + metallic sci-fi
- Subtle noise texture overlays
- Glowing borders and glow halos on key elements
- 3D models as hero and product visuals
- Metallic gradients on headings
